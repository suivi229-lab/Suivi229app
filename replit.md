# Suivi 229+

Application de gestion interne pour une entreprise de géolocalisation GPS (trackers véhicules).

## Stack
- React 18 + TypeScript + Vite
- Tailwind CSS
- Supabase (Auth + PostgreSQL)
- Lucide React

## Lancer le projet
```bash
npm run dev
```
Le serveur démarre sur le port 5000.

## User preferences
- **Toujours pousser sur GitHub (`git push origin main`) après chaque modification**, même mineure. L'utilisateur teste via son déploiement Render (lié à `github.com/suivi229-lab/Suivi229app`), pas via le preview Replit. Sans push, les changements ne sont pas visibles côté Render.
